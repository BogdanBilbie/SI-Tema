# SI-Tema

Prima data trebuie rulat main-ul din Nod KM, pentru ca am implementat serverul acolo
Rulam main=ul din Nod A, si astfel conectam clientul ce realizeaza criptarea
Rulam main-ul din Nod B, si astfel conectam clientul care primeste datele si le decripteaza
Pentru a cripta , scriem in consola specifica clientului din Nod A CBC sau OFB si asa alegem si modul de criptare
Key refresh-ul este setat sa se intample o data la 16 blocuri criptate trimis.
